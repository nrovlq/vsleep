vsleep
======

replacement of sleep(1) command for humans.
supports SUFFIX may be 's' for seconds (the default), 'm' for minutes, 'h' for hours or 'd' for days.


# EXAMPLE

~~~
$ vsleep 1m 
15:34:34 + 00:49 ETA 15:35:23 [===========.................................................]\
~~~
